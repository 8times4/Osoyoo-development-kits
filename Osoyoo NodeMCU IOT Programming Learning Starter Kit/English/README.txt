**************About this package**************

This package contains the basic tutorials of the Osoyoo NodeMCU Kits,and the related Arduino libraries,code,and videos.You can easily get started the Osoyoo Nodemcu Kit with this user manual in the root directory.

**************What can I get ?**************

***User manual of Osoyoo NodeMCU Kit
***Packing list of this kit
***Basic tutorials
***Arduino Code
***Arduino Libaries
***Tutorial videos
