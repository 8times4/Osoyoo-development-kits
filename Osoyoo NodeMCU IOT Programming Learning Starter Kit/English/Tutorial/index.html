Tutorial and sample projects(Arduino IDE)

This tutorial is about how to use the Osoyoo NodeMCU Kit with the Arduino IDE, it is designed to facilitate beginners and developers to quickly learn Arduino and IoT technology. It help users to turn the idea into a practical application and make the Internet of Things a reality. Note: If you are not so familiar with the Arduino, it's necessary for you to get more info about how to insatall the Arduino IDE and how to install an additional Arduino libraries before you built your first project.
